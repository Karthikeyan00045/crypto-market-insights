# Crypto Market Insights Dashboard

## 📌 Overview
This project connects to the **CoinMarketCap API**, extracts live cryptocurrency data, cleans it using Python, and visualizes insights in Tableau. The dashboard highlights market dynamics by grouping cryptocurrencies into **Dominant**, **Balance Providers**, and **Others**, with scenario analysis and annotated storytelling.

## 📊 Dataset
- **Source**: CoinMarketCap API  
- **Cleaning Process in Python**:
  - Removed duplicates and handled missing values  
  - Converted data types (string → float, date parsing)  
  - Validated consistency across records  
- **Output File**: `crypto_data_clean.csv`

## 🎨 Dashboard Design
- **Tool**: Tableau  
- **Features**:
  - Calculated fields for grouping  
  - Parameters for scenario analysis  
  - Dual-axis charts with conditional color encoding  
  - Narrative annotations for recruiter clarity  

## 💡 Key Insights
- Dominant cryptocurrencies drive overall market trends  
- Balance Providers stabilize volatility  
- Others highlight niche growth opportunities
